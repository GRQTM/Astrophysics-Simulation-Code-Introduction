## Code Profile

- **Method**: Moving-mesh finite-volume hydrodynamics
- **Core traits**: Voronoi moving mesh + gravity + (with modules) MHD/RT
- **Typical usage**: Cosmological galaxy-formation runs coupling gravity and hydrodynamics
- **Example Workflow**: `wave_1d`
