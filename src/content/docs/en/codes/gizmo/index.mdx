## Code Profile

- **Method**: Mesh-free finite-volume / finite-mass hydrodynamics
- **Core traits**: MFM/MFV hydro on moving particles + gravity modules
- **Typical usage**: Galaxy-scale and cosmological hydro with Lagrangian adaptivity
- **Example Workflow**: official hydro sanity case
