# Side-by-side code comparison

This table is intentionally compact for beginners.

| Code | Method family | Typical strength | Potential caveat |
| --- | --- | --- | --- |
| Arepo | Moving-mesh finite-volume | Interface tracking + adaptivity | Broad configuration space |
| Enzo | Eulerian AMR grid | Explicit refinement control | Refinement strategy sensitivity |
| Gadget | TreePM + SPH family | Mature cosmological particle workflows | SPH parameter dependence |
| Gizmo | Mesh-free finite-mass/volume | Lagrangian adaptivity with modern hydro modes | Mode/parameter tuning overhead |
| Ramses | Eulerian AMR finite-volume | Strong AMR-based multiphysics workflows | AMR criteria and runtime tuning complexity |
