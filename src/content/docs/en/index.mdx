## Goals for this guide

- Get every example running cleanly inside a `conda` virtual environment.
- Take at least one code all the way from **Quickstart to Production Run**.
- Not sure where to begin? Start with the [Arepo Quickstart](/en/codes/arepo/quickstart/).
