## Target outcome

- RAMSES 실행 파일을 빌드합니다.
- 소규모 AMR 유체 테스트를 실행합니다.
- 출력 파일과 실행 안정성을 확인합니다.

## Installation

1. RAMSES 공식 저장소를 클론합니다.
2. 플랫폼에 맞는 컴파일러/MPI/HDF5 환경을 구성합니다.
3. 현재 릴리스/브랜치의 사용자 가이드 절차로 빌드합니다.
4. 초기에는 최소 설정으로 시작한 뒤 모듈을 확장합니다.

## Example run used here

```bash
git clone https://github.com/ramses-organisation/ramses.git
cd ramses
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j8
mpirun -np 4 <path-to-ramses-binary> <input-namelist>
```

## 빠른 검증

- AMR 레벨 진화가 기대 동작과 일치하는지 확인합니다.
- 스냅샷/체크포인트 파일 생성을 확인합니다.
- 문서의 기준 곡선과 대표 프로파일을 비교합니다.
