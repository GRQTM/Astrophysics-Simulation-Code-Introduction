## Quick snapshot

- **Method family**: Eulerian AMR finite-volume hydrodynamics
- **Core traits**: AMR + N-body gravity + optional RT/MHD modules
- **Typical usage**: Cosmological and ISM simulations with adaptive grid refinement
- **Example used here**: RAMSES shock-tube style sanity case
