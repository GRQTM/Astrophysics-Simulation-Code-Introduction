## 코드 프로필

- **방법**: Eulerian AMR finite-volume hydrodynamics
- **핵심 특성**: AMR + N-body gravity + 선택적 RT/MHD 모듈
- **주요 용도**: 적응형 격자 세분화를 활용한 우주론적·ISM 시뮬레이션
- **기준 예제**: RAMSES shock-tube 검증 케이스
