# Gizmo

처음 읽는 사용자를 위한 foundation 단계 요약입니다.

## 빠른 스냅샷

- **방법 계열**: mesh-free finite-mass/finite-volume
- **주요 강점**: 입자 적응성과 현대적 hydro 옵션 결합
- **주의점**: 모드(MFM/MFV)와 파라미터 튜닝 필요
- **적합한 상황**: 입자 기반 유연성을 유지하며 hydro 품질을 높일 때
