# Gadget

처음 읽는 사용자를 위한 foundation 단계 요약입니다.

## 빠른 스냅샷

- **방법 계열**: TreePM 중력 + SPH 계열
- **주요 강점**: 우주론 분야 선행 워크플로우가 풍부함
- **주의점**: SPH 세부 설정 검증이 필수
- **적합한 상황**: 전통적 particle 기반 우주론 파이프라인
