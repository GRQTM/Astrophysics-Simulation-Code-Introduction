## 사용 시나리오 정의

- collisionless/hydro/결합 실행 중 목표 유형을 확정합니다.
- 주요 과학 지표와 필요한 출력 산출물을 정의합니다.
- 타깃 스케일을 입자 수/연화길이로 환산합니다.

## 수치 설정

- 목표 정확도에 맞춰 중력/적분 허용오차를 설정합니다.
- SPH 관련 파라미터는 민감도 테스트를 포함해 검증합니다.
- 소형 박스 파일럿으로 처리량과 안정성을 확인합니다.

## 데이터 계획

- 동역학 시간척도 기준으로 스냅샷 주기를 정합니다.
- 고빈도 경량 진단 채널을 별도로 운영합니다.
- 재시작 파일 포함 총 저장 용량을 산정합니다.
