## 문제 정의

- 과학 목표를 한 문장으로 명확히 정의합니다.
- 목표와 직접 연결되는 정량 진단량을 지정합니다.
- 계산 자원 투입 전에 성공 기준을 먼저 정합니다.

## 물리 모듈 구성

- 중력 + 유체 최소 구성부터 시작합니다.
- 냉각, MHD, RT, 피드백 모듈은 단계적으로 추가합니다.
- 모듈 변경 이력을 run manifest에 남깁니다.

## 도메인/해상도

- 물리 스케일에 맞춰 박스 크기와 해상도를 정합니다.
- 최소 1회 저해상도 파일럿으로 수렴 경향을 확인합니다.

## IC/출력 계획

- IC 파일 및 생성 스크립트 버전을 고정합니다.
- 대용량 스냅샷과 경량 진단 출력 주기를 분리합니다.
- 재시작 파일 포함 저장 용량을 사전 확보합니다.
