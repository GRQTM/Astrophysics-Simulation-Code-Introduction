## Target outcome

- GIZMO 실행 파일을 컴파일합니다.
- MFM/MFV 기반 기본 hydro 예제를 실행합니다.
- 출력 및 로그가 정상인지 확인합니다.

## Installation

1. 빌드 모드에 필요한 컴파일러/MPI/라이브러리를 준비합니다.
2. 공개 저장소를 클론하고 빌드 문서를 확인합니다.
3. 첫 테스트용 최소 `Config.sh`를 선택합니다.
4. 컴파일 후 실행 파일 생성 여부를 검증합니다.

## Example run used here

```bash
git clone https://github.com/pfhopkins/gizmo-public.git
cd gizmo-public
cp <example-config> ./Config.sh
make -j8
mpirun -np 4 ./GIZMO <example-parameter-file>
```

## 빠른 검증

- 로그에서 비정상 안정성 경고가 없는지 확인합니다.
- 스냅샷/진단 파일이 정상 생성되는지 확인합니다.
- 기준 테스트와 대표 프로파일을 비교합니다.
