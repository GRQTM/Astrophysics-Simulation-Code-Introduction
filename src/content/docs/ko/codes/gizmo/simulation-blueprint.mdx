## 과학 목표 설정

- 핵심 관측량과 허용 오차를 먼저 정합니다.
- MFM/MFV 중 주력 hydro 모드를 결정합니다.
- 반드시 모니터링할 진단량을 사전에 확정합니다.

## 물리 모델 구성

- 중력 + 유체 기준선을 먼저 맞춥니다.
- 냉각, 별형성, 피드백, MHD는 순차적으로 추가합니다.
- 설정 변경을 run registry에 누적 기록합니다.

## 해상도/출력 계획

- 입자 질량/연화길이를 물리 목표에 맞춰 설정합니다.
- 동역학 시간척도 기준으로 스냅샷 주기를 정합니다.
- 고빈도 경량 진단 채널을 별도로 둡니다.
