## 과학 목표 설정

- 핵심 관측량과 허용 오차를 먼저 정합니다.
- MFM / MFV 중 주력 hydro 모드를 결정합니다.
- 실행 전에 반드시 모니터링할 진단량을 확정합니다.

## 물리 모델 구성

- 중력 + 유체역학 기준선을 먼저 구성합니다.
- 냉각, 별 형성, 피드백, MHD 모듈은 하나씩 순서대로 추가합니다.
- 설정 변경 내역을 run registry에 기록해 재현성을 유지합니다.

## 해상도와 출력 계획

- 물리 목표에 맞춰 입자 질량과 연화길이를 설정합니다.
- 동역학 시간척도 기준으로 스냅샷 출력 주기를 정합니다.
- QA용 고빈도 경량 진단 채널을 별도로 둡니다.

## 본실행 전 점검

- 수렴성 확인을 위한 짧은 파일럿을 완료합니다.
- 재시작 동작과 출력 포맷 호환성을 확인합니다.
- 전체 캠페인에 필요한 walltime과 저장 공간을 추정합니다.
