## 코드 프로필

- **방법**: Mesh-free finite-volume / finite-mass hydrodynamics
- **핵심 특성**: 움직이는 입자 위에서 MFM/MFV hydro + 중력 모듈
- **주요 용도**: 은하 규모 및 우주론적 유체 시뮬레이션 (Lagrangian 적응형)
- **기준 예제**: 공식 hydro sanity case
