# 처음부터 배우는 천체물리 시뮬레이션

이 사이트는 네 가지 주요 코드(Arepo, Enzo, Gadget, Gizmo)를 입문자 눈높이에서 소개합니다.

## 추천 학습 순서

1. 먼저 [기초](/ko/basics/)를 읽습니다.
2. 코드별 페이지를 봅니다: [Arepo](/ko/codes/arepo/), [Enzo](/ko/codes/enzo/), [Gadget](/ko/codes/gadget/), [Gizmo](/ko/codes/gizmo/)
3. [비교](/ko/compare/)와 [용어집](/ko/glossary/)으로 맥락을 정리합니다.

## 현재 릴리스 범위

- 지원 언어: 영어, 한국어
- 콘텐츠 원칙: 영어 원문을 기준으로 한국어 번역 유지
- 정확도 원칙: 공식 문서, 논문, 버전 정보 기반으로 확장
