# 코드 한눈에 비교하기

어떤 코드를 먼저 써 볼지 고민된다면, 아래 표에서 핵심 차이를 빠르게 파악해 보세요.

## 핵심 비교

| 코드 | 방법 | 장점 | 유의점 |
| --- | --- | --- | --- |
| Arepo | Moving-mesh | 가스와 함께 움직이는 Voronoi 메시로 유체 흐름과 형태를 정밀하게 추적합니다. | 초기 설정이 다소 복잡할 수 있습니다. |
| Enzo | Eulerian AMR | 충격파를 선명하게 포착하고, 필요한 영역만 집중적으로 고해상도로 풀 수 있습니다. | 빠르게 이동하는 가스 덩어리를 추적하기에는 조금 불리합니다. |
| Gadget | SPH | 빈 공간을 효율적으로 처리하며, 사용자 커뮤니티가 가장 넓습니다. | 격자 기반 코드에 비해 충격파 해상도가 약간 부드럽습니다. |
| Gizmo | Mesh-free (MFM/MFV) | SPH와 격자 방식의 장점을 결합해 높은 정확도를 제공합니다. | 처음 파라미터를 맞추는 과정에 세심한 조정이 필요합니다. |
| Ramses | Eulerian AMR | 대규모 HPC 클러스터에서 뛰어난 확장성과 속도를 보여 줍니다. | 출력 데이터가 커질 수 있어 스토리지 관리가 필요합니다. |

> **실전 팁:** 처음부터 "완벽한" 코드를 고를 필요는 없습니다. 소속 연구실에서 주로 쓰는 코드나, 지금 환경에서 가장 설치하기 쉬운 코드부터 시작해 보세요.
