# 코드 비교 한눈에 보기

입문자 기준으로 핵심만 압축한 표입니다.

| 코드 | 방법 계열 | 대표 강점 | 잠재적 주의점 |
| --- | --- | --- | --- |
| Arepo | Moving-mesh 유한체적 | 경계면 추적 + 적응성 | 설정 조합이 많아 초기 복잡도 큼 |
| Enzo | Eulerian AMR 격자 | refinement 제어가 명시적 | refinement 전략 민감도 |
| Gadget | TreePM + SPH 계열 | 우주론 입자 시뮬레이션의 성숙한 워크플로 | SPH 파라미터 의존성 |
| Gizmo | Mesh-free finite-mass/volume | 라그랑지안 적응성과 현대적 hydro 모드 | 모드/파라미터 튜닝 부담 |
| Ramses | Eulerian AMR 유한체적 | AMR 기반 다중물리 확장성 | AMR 기준/런타임 튜닝 복잡도 |
