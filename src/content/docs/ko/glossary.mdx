# 핵심 시뮬레이션 용어

다국어 품질을 위해 원문 용어 1개에 권장 번역 1개를 매핑해 관리합니다.

## 용어

- **Hydrodynamics**: 유체 흐름, 충격파, 혼합을 계산하는 수치 방법 전반.
- **N-body**: 다수 입자의 중력 상호작용을 시간에 따라 계산하는 문제.
- **AMR (Adaptive Mesh Refinement)**: 필요한 영역만 선택적으로 격자를 세분화하는 기법.
- **SPH (Smoothed Particle Hydrodynamics)**: 커널 가중 이웃으로 물리량을 근사하는 입자 기반 유체 모델.
- **Resolution**: 수치적으로 표현 가능한 최소 스케일.
- **Initial conditions**: 결과 해석을 크게 좌우하는 시작 상태.

## 운영 원칙

- 영어 용어를 source-of-truth 키로 관리
- 한국어 및 향후 언어 번역을 동일 키에 매핑
